=== WooCommerce Offline Invoices Request ===
Contributors: amglabs
Requires at least: 4.6
Stable tag: 1.0
Tested up to: 4.6

This plugin allows customers to request offline invoices. Invoices must be generated and sent to customers by shop admin.
